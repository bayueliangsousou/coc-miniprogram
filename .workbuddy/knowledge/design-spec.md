# 小程序端设计规范

> 本文档记录小程序端（coc-miniprogram）的设计规范数值，后续功能开发须严格遵守，不得新造规则数值。

## 色彩体系

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#C9A84C` | 金色，用于按钮、图标、高亮边框 |
| 主色深色 | `#A67C2F` | 主色 hover/active 状态 |
| 背景深色 | `#1A1A1A` | 页面主背景 |
| 背景中色 | `#2A2A2A` | 卡片/弹窗背景 |
| 背景浅色 | `#3A3A3A` | 输入框/列表项背景 |
| 文字主色 | `#E0E0E0` | 主要文字 |
| 文字次色 | `#999999` | 次要文字、占位符 |
| 成功色 | `#4CAF50` | 成功提示 |
| 错误色 | `#DC2626` | 错误提示、删除操作 |
| 警告色 | `#FF9800` | 警告提示 |
| 边框色 | `#444444` | 默认边框 |
| 边框高亮 | `#C9A84C` | 聚焦/选中状态边框 |

## 间距规范

| 层级 | 数值（rpx） | 说明 |
|------|------------|------|
| 页面边距 | 32rpx | 页面左右留白 |
| 卡片内边距 | 24rpx | 卡片内容区 padding |
| 卡片圆角 | 16-24rpx | 卡片/弹窗 border-radius |
| 按钮内边距 | 16rpx 32rpx | 按钮文字周围留白 |
| 列表项间距 | 16rpx | 列表项之间的 gap |
| 表单项间距 | 24rpx | 表单字段之间的间距 |

## 字号规范

| 用途 | 数值（rpx） | 说明 |
|------|------------|------|
| 页面标题 | 40rpx | 页面顶部主标题 |
| 卡片标题 | 32rpx | 卡片/弹窗标题 |
| 正文 | 28rpx | 普通正文文字 |
| 次要文字 | 24rpx | 描述、提示文字 |
| 小字 | 20rpx | 时间戳、标签文字 |

|------|------------|------------|------|

| 次按钮 | 72rpx | 12rpx | 次要操作按钮 |
| 小按钮 | 56rpx | 8rpx | 紧凑型按钮 |
