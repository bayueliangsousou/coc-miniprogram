# 武器模块更新说明

## 更新内容

### 1. 武器模块展示逻辑
- **未编辑技能前**：武器模块完全不展示
- **编辑技能后**：武器模块出现，显示"添加武器"按钮
- **点击添加武器**：在当前位置展开武器输入表单
- **保存角色卡后**：武器模块变为下拉选择展示模式

### 2. 添加武器流程
1. 点击"添加武器"按钮
2. **展开武器输入表单**：
   - **第一行**：战斗技能下拉选择器（显示技能名称和鉴定值）
   - **第二行**：武器名称输入框 + 伤害值输入框
3. 点击"确认添加"保存武器
4. 可以继续点击"添加武器"添加更多武器

### 3. 武器数据结构
```javascript
{
  name: "柯尔特M1911",        // 武器名称
  skillKey: "手枪",           // 技能名称
  skillValue: 50,             // 技能值
  skillDisplay: "手枪 50%",   // 展示文本
  damage: "1d8+2"             // 伤害值
}
```

## 界面流程

```
新建角色
    ↓
武器模块不显示
    ↓
点击"技能编辑" → 分配技能点
    ↓
返回角色编辑页
    ↓
武器模块显示（有"添加武器"按钮）
    ↓
点击"添加武器"
    ↓
展开武器输入表单：
┌─────────────────────────┐
│ 战斗技能 [下拉选择]      │  ← 显示"手枪 50%"
├─────────────────────────┤
│ 武器名称 [输入框] │ 伤害值 [输入框] │
├─────────────────────────┤
│ [取消] [确认添加]        │
└─────────────────────────┘
    ↓
确认添加 → 武器加入列表
    ↓
可以继续添加更多武器
    ↓
保存角色卡
    ↓
武器模块变为下拉选择展示模式
```
